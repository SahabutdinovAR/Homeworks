## 04

```
1. Реализовать функцию:

pubic static int parse(int array[]);

На вход функция принимает массив из целых положительных чисел, состоящих из одной цифры. Результатом выполнения данной функции является число, полученное на основе массива путем склеивания всех цифр.

Например:

int array[] = {4, 5, 7};

int result = parse(array); // result == 457

2. Реализовать программу (набор функций и процедур на усмотрение разработчика):

Программа принимает на вход последовательность чисел (числа в диапазоне от -1000 до 1000) и выводит число, которое встречается чаще остальных. Если таких чисел несколько, вывести последнее.

Например:
35
36
36
35
36

Ответ: 36
```
